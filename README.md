# Salesforce Postman API Examples

A practical project created to demonstrate how to configure and use Salesforce REST APIs with Postman.

The goal of this repository is to provide a simple reference for beginners who want to learn how to authenticate, configure a Postman environment, execute API requests, and explore Salesforce objects using REST APIs.

## Contents

### Postman Collection

The Postman collection includes examples for:

* Authentication
* Creating records
* Querying records
* Exploring Salesforce objects
* Basic API testing

### Environment Example

An example Postman environment is provided to help configure:

* Base URL
* Client ID
* Client Secret
* Username
* Password
* Security Token

Replace the placeholder values with your own Salesforce credentials.

## Setup Guide

Follow the instructions in:

```text
docs/setup-guide.md
```

This guide explains how to:

1. Create a Salesforce Developer Account
2. Create a Connected App
3. Configure OAuth settings
4. Configure Postman variables
5. Generate an Access Token
6. Execute API requests
